# socket

## 2.1 常用API

Java提供了不同的API，其提供的网络功能有四大类：

```
InetAddress:用于标识网络上的硬件资源，主要是IP地址
URL：统一资源定位符，通过URL可以直接读取或写入网络上的数据
Sockets：使用TCP协议实现的网络通信Socket相关的类
Datagram:使用UDP协议，将数据保存在用户数据报中，通过网络进行通信。
```
